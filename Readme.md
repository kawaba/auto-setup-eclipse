# Eclipse 自動セットアップ手順（Markdown版）

これらのスクリプトと設定ファイルは、次のことを行います。

1. **Eclipse 2025-09 をダウンロードして展開する**  
2. **JDK 25 をダウンロードして Eclipse に組み込む**  
3. **Pleiades をダウンロードして展開し、Eclipse を日本語化する**  
4. **Spring Boot 開発に必要なプラグインをインストールする**  
5. **GitHub Copilot プラグインをインストールし、AI を利用可能にする**  
6. **Eclipse のフォントや構文色を設定する**

---

## 実行手順

### (1) setup.bat を実行する
まず最初に `setup.bat` を実行します。

---

### (2) セットアップ済み Eclipse の初回起動
`setup.bat` によりセットアップされた Eclipse を起動します。

- 起動方法：  
  `eclipse/eclipse.exe` をダブルクリック  
- ランチャーで **ワークスペースに `../workspace` を指定**  
- Eclipse が完全に起動したら **何もせず終了**

---

### (3) plugins.bat を実行する
- `plugins.bat` を実行すると、内部で `plugins.ps1` が動きます。
- このスクリプトは Eclipse に必要なプラグインをインストールします。
- 処理の最後で、不要なファイル・ディレクトリのクリーンアップを行います。

最終的に残るフォルダ：

- `eclipse`  
- `workspace`  

また、`eclipse` フォルダにはフォント設定用の  
**`font-color-compiler-java25.epf`** がコピーされます。

---

### (4) Eclipse のフォント・構文色の設定

スクリプト完了後、もう一度 Eclipse を起動します。

- ランチャーで **ワークスペースに `../workspace` を指定**

#### 設定インポート手順

1. メニューから **ファイル → インポート** を選択  
2. **一般 → 設定 → ［次へ］**  
3. 「ソース設定ファイル」欄の **参照** ボタンを押す  
4. `eclipse/font-color-compiler-java25.epf` を選択  
5. **［完了］** を押す  
6. 再起動を求められたら **［再開］** を押す

---

## 注意事項

- 既存のワークスペースを使う場合  
  - ランチャーで既存ワークスペースを指定  
  - 起動後に **１〜６の設定インポート操作** を行う  
  - 設定はワークスペースに保存されるため、  
    **異なるワークスペースを使うたびに (4) の設定を繰り返すと良い**
